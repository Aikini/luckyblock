Lucky Block for Minecraft 1.3 Add-on changes
The following assumes you have an add-on named "Lucky Block Red"

----

=== ASSETS ===

assets/blockstates/lucky_block_red.json
    Change variant "normal" to ""
    Change "lucky:lucky_block_red" to "lucky:block/lucky_block_red"

assets/lang/en_US.lang
    Change to json format
    Change block and item name formats, e.g.
        tile.luckyBlockRed.name -> block.lucky.lucky_block_red
        item.luckySwordRed.name -> item.lucky.lucky_sword_red

data/lucky/recipes/lucky_block_red.json
    Create this file to add a custom crafting recipe
    Use the default lucky block recipe file as an example
